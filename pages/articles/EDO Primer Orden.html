1. Definición y Conceptos Básicos

Una ecuación diferencial ordinaria (EDO) de primer orden es una ecuación que involucra una función desconocida \(y(x)\) y su primera derivada \(y'(x)\). Tiene la forma general:

\(F(x, y, y') = 0\) o equivalentemente \(y' = f(x, y)\)

Donde:

  • \(x\) es la variable independiente
  • \(y = y(x)\) es la función desconocida (variable dependiente)
  • \(y' = \frac{dy}{dx}\) es la derivada de \(y\) con respecto a \(x\)
  • \(f(x, y)\) es una función dada de dos variables

Objetivo Principal

Encontrar todas las funciones \(y(x)\) que satisfacen la ecuación diferencial. La solución general suele contener una constante arbitraria \(C\), ya que al integrar una vez obtenemos una familia de soluciones.

Ejemplos de EDO de Primer Orden

EDO más simple
\(y' = 2x\)

Solución: Integrando ambos lados, \(y = x^2 + C\)

Con función de y
\(y' = y\)

Solución: \(y = Ce^x\) (crecimiento exponencial)

Con ambas variables
\(y' = x + y\)

Solución: \(y = Ce^x - x - 1\)

2. Métodos de Solución para EDO de Primer Orden

Existen varios métodos para resolver EDO de primer orden, dependiendo de la forma que tenga la ecuación. Los principales son:

2.1 Variables Separables

Una EDO es separable si puede escribirse en la forma:

\(\frac{dy}{dx} = g(x)h(y)\) o \(\frac{dy}{dx} = \frac{g(x)}{h(y)}\)

Procedimiento:

  1. Separar variables: \(\frac{dy}{h(y)} = g(x)dx\)
  2. Integrar ambos lados: \(\int \frac{dy}{h(y)} = \int g(x)dx\)
  3. Resolver las integrales
  4. Despejar \(y\) si es posible (solución explícita) o dejar en forma implícita
Ejemplo: Variables Separables

Resolver: \(\frac{dy}{dx} = x y^2\)

  1. Separar: \(\frac{dy}{y^2} = x dx\)
  2. Integrar: \(\int y^{-2} dy = \int x dx\)
  3. Resultado: \(-y^{-1} = \frac{x^2}{2} + C\)
  4. Solución: \(y = -\frac{1}{\frac{x^2}{2} + C} = -\frac{2}{x^2 + 2C}\)

2.2 Ecuaciones Exactas

Una EDO de la forma \(M(x,y)dx + N(x,y)dy = 0\) es exacta si existe una función \(F(x,y)\) tal que:

\(\frac{\partial F}{\partial x} = M(x,y)\) y \(\frac{\partial F}{\partial y} = N(x,y)\)

Condición de exactitud: \(\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}\)

2.3 Factor Integrante

Si una ecuación no es exacta, a veces podemos multiplicarla por un factor integrante \(\mu(x)\) o \(\mu(y)\) para convertirla en exacta.

Para ecuaciones lineales: \(\mu(x) = e^{\int P(x)dx}\) donde \(y' + P(x)y = Q(x)\)

2.4 Ecuaciones Lineales

Una EDO de primer orden es lineal si puede escribirse en la forma:

\(y' + P(x)y = Q(x)\)

Solución: \(y = e^{-\int P(x)dx} \left[ \int e^{\int P(x)dx} Q(x) dx + C \right]\)

Tipo de EDO Forma Método de Solución
Variables separables \(y' = g(x)h(y)\) Separar variables e integrar
Exactas \(Mdx + Ndy = 0\) con \(M_y = N_x\) Encontrar \(F\) tal que \(F_x = M\), \(F_y = N\)
Lineales \(y' + P(x)y = Q(x)\) Factor integrante \(\mu = e^{\int P dx}\)
Bernoulli \(y' + P(x)y = Q(x)y^n\) Sustitución \(v = y^{1-n}\)

3. Campos de Dirección (Método Gráfico)

Un campo de dirección es una representación gráfica de una ecuación diferencial de primer orden. En cada punto \((x, y)\) del plano, se dibuja un pequeño segmento de recta con pendiente \(f(x, y)\).

¿Cómo interpretar un campo de dirección?

  • Cada segmento muestra la dirección que tomaría una solución que pase por ese punto.
  • Las soluciones de la EDO son curvas que siguen la dirección indicada por los segmentos.
  • Los campos de dirección son útiles cuando no podemos resolver la EDO analíticamente.
  • Nos permiten visualizar el comportamiento de las soluciones sin calcularlas explícitamente.

4. Método de Euler (Solución Numérica)

El método de Euler es un procedimiento numérico para aproximar soluciones de ecuaciones diferenciales. Es especialmente útil cuando no podemos encontrar una solución analítica.

Fórmula de Euler: \(y_{n+1} = y_n + h \cdot f(x_n, y_n)\)

Donde:

  • \(h\) es el tamaño del paso (step size)
  • \(x_{n+1} = x_n + h\)
  • \(f(x_n, y_n)\) es la pendiente en el punto \((x_n, y_n)\)

5. Ejemplos Resueltos Paso a Paso

Ejemplo 1: Variables Separables

Problema: Resolver \(\frac{dy}{dx} = \frac{x}{y}\), con \(y(0) = 2\)

  1. Separar variables: \(y \, dy = x \, dx\)
  2. Integrar ambos lados: \(\int y \, dy = \int x \, dx\)
  3. Resultado: \(\frac{y^2}{2} = \frac{x^2}{2} + C\)
  4. Simplificar: \(y^2 = x^2 + 2C\)
  5. Usar condición inicial \(y(0)=2\): \(4 = 0 + 2C \Rightarrow C = 2\)
  6. Solución particular: \(y^2 = x^2 + 4\)
\(y = \sqrt{x^2 + 4}\) (tomamos la raíz positiva por la condición inicial)
Ejemplo 2: Ecuación Lineal

Problema: Resolver \(y' + 2xy = x\), con \(y(0) = 1\)

  1. Identificar \(P(x)\) y \(Q(x)\): \(P(x) = 2x\), \(Q(x) = x\)
  2. Calcular factor integrante: \(\mu(x) = e^{\int 2x dx} = e^{x^2}\)
  3. Multiplicar la EDO por \(\mu(x)\): \(e^{x^2}y' + 2xe^{x^2}y = xe^{x^2}\)
  4. Reconocer derivada: \(\frac{d}{dx}(e^{x^2}y) = xe^{x^2}\)
  5. Integrar: \(e^{x^2}y = \int xe^{x^2} dx = \frac{1}{2}e^{x^2} + C\)
  6. Despejar \(y\): \(y = \frac{1}{2} + Ce^{-x^2}\)
  7. Usar condición inicial \(y(0)=1\): \(1 = \frac{1}{2} + C \Rightarrow C = \frac{1}{2}\)
Solución: \(y = \frac{1}{2} + \frac{1}{2}e^{-x^2} = \frac{1}{2}(1 + e^{-x^2})\)
Ejemplo 3: Ecuación de Bernoulli

Problema: Resolver \(y' + y = xy^2\)

  1. Identificar como Bernoulli: \(y' + y = xy^2\), \(n=2\)
  2. Sustitución: \(v = y^{1-2} = y^{-1}\)
  3. Derivar: \(v' = -y^{-2}y'\)
  4. Sustituir en la EDO: \(-y^2 v' + y = xy^2\)
  5. Dividir entre \(-y^2\): \(v' - y^{-1} = -x\)
  6. Recordar \(v = y^{-1}\): \(v' - v = -x\)
  7. Resolver EDO lineal en \(v\): \(v = 1 + x + Ce^x\)
  8. Volver a \(y\): \(y^{-1} = 1 + x + Ce^x\)
Solución: \(y = \frac{1}{1 + x + Ce^x}\)
