1. Aritmética

1.1 Números Enteros y Racionales

Los números enteros incluyen todos los números positivos, negativos y el cero. Son fundamentales para realizar operaciones básicas en matemáticas.

💡 Concepto Clave

Los números racionales son aquellos que pueden expresarse como una fracción a/b, donde a y b son números enteros y b ≠ 0.

Propiedades de los números enteros:

  • Propiedad conmutativa: a + b = b + a
  • Propiedad asociativa: (a + b) + c = a + (b + c)
  • Elemento neutro: a + 0 = a
  • Elemento opuesto: a + (-a) = 0

📝 Ejemplo 1: Operaciones con enteros

Problema: Calcular (-5) + 8 - (-3)

Solución:

Paso 1: (-5) + 8 - (-3)

Paso 2: (-5) + 8 + 3

Paso 3: 3 + 3

Respuesta: 6

1.2 Fracciones y Decimales

Las fracciones representan partes de un todo. Pueden convertirse a decimales mediante división.

Fracción a decimal: a/b = a ÷ b

Operaciones con fracciones:

  • Suma: a/b + c/d = (ad + bc) / bd
  • Resta: a/b - c/d = (ad - bc) / bd
  • Multiplicación: a/b × c/d = (ac) / (bd)
  • División: a/b ÷ c/d = (ad) / (bc)

📝 Ejemplo 2: Suma de fracciones

Problema: Calcular 2/3 + 1/4

Solución:

Paso 1: Encontrar el MCM de 3 y 4 = 12

Paso 2: (2×4)/(3×4) + (1×3)/(4×3)

Paso 3: 8/12 + 3/12

Paso 4: 11/12

Respuesta: 11/12

1.3 Porcentajes

Un porcentaje representa una cantidad como fracción de 100.

Porcentaje: x% = x/100

⚠️ Importante

Para calcular el porcentaje de una cantidad, multiplica la cantidad por el porcentaje dividido entre 100.

2. Álgebra Básica

2.1 Expresiones Algebraicas

Una expresión algebraica es una combinación de números, variables y operaciones matemáticas.

Términos importantes:

  • Variable: Símbolo que representa un número desconocido (x, y, z)
  • Coeficiente: Número que multiplica a la variable
  • Término: Producto de un coeficiente y variables
  • Término semejante: Términos con las mismas variables y exponentes

📝 Ejemplo 3: Simplificación de expresiones

Problema: Simplificar 3x + 5x - 2x

Solución:

Paso 1: Identificar términos semejantes (todos tienen x)

Paso 2: Sumar/restar coeficientes: (3 + 5 - 2)x

Respuesta: 6x

2.2 Ecuaciones de Primer Grado

Una ecuación de primer grado es una igualdad que contiene una variable elevada a la primera potencia.

Forma general: ax + b = 0

Pasos para resolver ecuaciones:

  1. Simplificar ambos lados de la ecuación
  2. Agrupar términos con variables a un lado
  3. Agrupar términos constantes al otro lado
  4. Despejar la variable

📝 Ejemplo 4: Resolver ecuación

Problema: Resolver 3x + 5 = 14

Solución:

Paso 1: 3x + 5 = 14

Paso 2: 3x = 14 - 5

Paso 3: 3x = 9

Paso 4: x = 9/3

Respuesta: x = 3

2.3 Productos Notables

Son productos de expresiones algebraicas que siguen patrones específicos y pueden resolverse rápidamente.

Principales productos notables:

(a + b)² = a² + 2ab + b²
(a - b)² = a² - 2ab + b²
(a + b)(a - b) = a² - b²

✅ Consejo de Estudio

Memoriza estos productos notables. Te ahorrarán mucho tiempo en exámenes y te ayudarán a factorizar más rápido.

3. Geometría

3.1 Figuras Planas

Triángulos

Un triángulo es un polígono de tres lados. La suma de sus ángulos internos siempre es 180°.

Área = (base × altura) / 2

Tipos de triángulos:

  • Por sus lados: Equilátero, Isósceles, Escaleno
  • Por sus ángulos: Rectángulo, Obtusángulo, Acutángulo

Cuadriláteros

Polígonos de cuatro lados. La suma de sus ángulos internos es 360°.

Área del rectángulo = base × altura
Área del cuadrado = lado²

3.2 Teorema de Pitágoras

En todo triángulo rectángulo, el cuadrado de la hipotenusa es igual a la suma de los cuadrados de los catetos.

c² = a² + b²

📝 Ejemplo 5: Aplicación del Teorema de Pitágoras

Problema: Un triángulo rectángulo tiene catetos de 3 cm y 4 cm. ¿Cuál es la longitud de la hipotenusa?

Solución:

Paso 1: c² = 3² + 4²

Paso 2: c² = 9 + 16

Paso 3: c² = 25

Paso 4: c = √25

Respuesta: c = 5 cm

3.3 Circunferencia y Círculo

La circunferencia es el conjunto de puntos equidistantes de un punto llamado centro.

Perímetro = 2πr
Área = πr²

💡 Dato importante

π (pi) es aproximadamente 3.14159. En la mayoría de problemas puedes usar π ≈ 3.14
