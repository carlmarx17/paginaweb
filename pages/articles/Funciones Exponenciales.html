Ejercicio 2: Hallar la función exponencial

La gráfica de una función exponencial de la forma f(x) = a·bx pasa por el punto P. Determina la expresión de la función (valor de a y b) e indica si es creciente o decreciente.

SOLUCIÓN

Selecciona un inciso para ver su solución paso a paso

i) ¿Por qué ningún punto P puede tener y = 0?

Explica por qué no se puede indicar ningún punto P cuyo valor de y sea 0.

Solución: En una función exponencial de la forma f(x) = a·bx (con a ≠ 0 y b > 0, b ≠ 1), el valor de f(x) nunca puede ser 0 porque bx > 0 para todo x real. Por lo tanto, a·bx ≠ 0. La gráfica se acerca al eje X (asíntota horizontal) pero nunca lo cruza.

Más Ayuda para Resolver

Consejos y estrategias para resolver los ejercicios de funciones exponenciales

📈 Cómo determinar si es creciente o decreciente

  • ✓ Si b > 1: La función es creciente (sube)
  • ✓ Si 0 < b < 1: La función es decreciente (baja)
  • ✓ Ejemplo: f(x) = 2x → b=2>1 → Creciente

🔢 Cómo encontrar a y b dado un punto P(x, y)

  1. Sustituye x e y en y = a·bx
  2. Si tienes un segundo punto, plantea otro ecuación
  3. Resuelve el sistema de ecuaciones
  4. Verifica si b > 1 (creciente) o 0 < b < 1 (decreciente)

Ejemplo: P(2, 9) con a=1

9 = 1·b2 → b2 = 9 → b = 3 (b>1 → creciente)

🎯 Contextos Reales para Funciones Exponenciales

Crecimiento Poblacional

P(t) = P₀·ert donde r es la tasa de crecimiento

Interés Compuesto

A(t) = P·(1 + r/n)nt donde P es el capital inicial

Desintegración Radiactiva

N(t) = N₀·(1/2)t/T donde T es la vida media

Propagación de Virus

I(t) = I₀·R₀t donde R₀ es el número reproductivo básico

📊 Cómo esbozar la gráfica

  • 1. Calcula f(0) = a + c (intersección con eje Y)
  • 2. Encuentra 2-3 puntos más (ej: f(1), f(-1), f(2))
  • 3. Determina la asíntota horizontal y = c
  • 4. Une los puntos con una curva suave

Solución Ejercicio 2 (Ejemplos)

Algunos incisos resueltos

a) P(2, 4) con a=1

Resolución:

4 = 1·b2 → b2 = 4 → b = 2 (tomamos b>0)

Función: f(x) = 2x

Comportamiento: Creciente (b=2>1)

b) P(2, 1/4) con a=1

Resolución:

1/4 = 1·b2 → b2 = 1/4 → b = 1/2

Función: f(x) = (1/2)x

Comportamiento: Decreciente (b=0.5<1)

i) ¿Por qué y nunca es 0?

Explicación: En f(x) = a·bx (con a≠0, b>0, b≠1), bx siempre es positivo. Por lo tanto, a·bx nunca puede ser 0. La gráfica tiene al eje X como asíntota horizontal.
