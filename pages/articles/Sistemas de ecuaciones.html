Método de Sustitución

Despejá una variable en una ecuación y sustituí su valor en la otra. Ideal cuando una variable ya está despejada.

Fórmula: y = mx + b

⚖️

Método de Igualación

Despejá la misma variable en ambas ecuaciones e igualá las expresiones. Perfecto cuando ambas son fáciles de despejar.

y₁ = y₂

➕➖

Método de Reducción

Sumá o restá las ecuaciones para eliminar una variable. Excelente cuando los coeficientes son opuestos o fácilmente igualables.

E₁ + E₂ = E₃

Tipos de Sistemas de Ecuaciones

No todos los sistemas tienen una solución única. Dependiendo de cómo se relacionan las ecuaciones, podemos tener diferentes casos.

🎯

Solución Única

Las rectas se intersecan en un solo punto. El sistema es compatible determinado.

2x + y = 5
x - y = 1
∞

Infinitas Soluciones

Las rectas son coincidentes (la misma línea). El sistema es compatible indeterminado.

2x + 2y = 4
x + y = 2
🚫

Sin Solución

Las rectas son paralelas y no se intersecan. El sistema es incompatible.

2x + y = 3
2x + y = 5

Métodos de Resolución Paso a Paso

🔄

Método de Sustitución

Cuando una variable es fácil de despejar

Paso 1: Despejá una variable en una de las ecuaciones.
Paso 2: Sustituí la expresión obtenida en la otra ecuación.
Paso 3: Resolvé la ecuación resultante (con una sola variable).
Paso 4: Sustituí el valor encontrado en la expresión del paso 1.
Ejemplo: Resolver: 2x + y = 7, x - y = 2

1. Despejo y en la primera: y = 7 - 2x
2. Sustituyo en la segunda: x - (7 - 2x) = 2
3. Resuelvo: x - 7 + 2x = 2 → 3x = 9 → x = 3
4. Encuentro y: y = 7 - 2(3) = 1
Solución: x = 3, y = 1
⚖️

Método de Igualación

Cuando ambas variables son fáciles de despejar

Paso 1: Despejá la misma variable en ambas ecuaciones.
Paso 2: Igualá las dos expresiones obtenidas.
Paso 3: Resolvé la ecuación resultante.
Paso 4: Sustituí el valor encontrado en cualquiera de las expresiones del paso 1.
Ejemplo: Resolver: 3x + 2y = 8, x - y = 1

1. Despejo x en ambas:
  De la primera: x = (8 - 2y)/3
  De la segunda: x = y + 1
2. Igualo: (8 - 2y)/3 = y + 1
3. Resuelvo: 8 - 2y = 3y + 3 → 5 = 5y → y = 1
4. Encuentro x: x = 1 + 1 = 2
Solución: x = 2, y = 1
➕➖

Método de Reducción

Cuando los coeficientes son fácilmente igualables

Paso 1: Multiplicá una o ambas ecuaciones para igualar coeficientes de una variable.
Paso 2: Sumá o restá las ecuaciones para eliminar esa variable.
Paso 3: Resolvé la ecuación resultante.
Paso 4: Sustituí el valor encontrado en cualquiera de las ecuaciones originales.
Ejemplo: Resolver: 2x + 3y = 7, 3x - 2y = 4

1. Multiplico la primera por 2 y la segunda por 3:
  4x + 6y = 14
  9x - 6y = 12
2. Sumo ambas: 13x = 26
3. Resuelvo: x = 2
4. Encuentro y: 2(2) + 3y = 7 → 3y = 3 → y = 1
Solución: x = 2, y = 1

Aplicaciones en la Vida Real

Los sistemas de ecuaciones no son solo ejercicios de libro. Se usan en muchas situaciones prácticas:

💰

Problemas de Costos y Precios

Determinar precios unitarios a partir de compras combinadas

Ejemplo: Si 3 manzanas y 2 naranjas cuestan $5, y 2 manzanas y 3 naranjas cuestan $6, ¿cuánto cuesta cada fruta?

3m + 2n = 5
2m + 3n = 6
📊

Mezclas y Soluciones

Calcular cantidades para obtener mezclas con características específicas

Ejemplo: Mezclar dos soluciones con diferentes concentraciones para obtener una concentración deseada.

x + y = 100 (volumen total)
0.3x + 0.5y = 0.4*100 (concentración)
🚗

Problemas de Velocidad y Tiempo

Encontrar velocidades individuales a partir de situaciones combinadas

Ejemplo: Dos vehículos que viajan en direcciones opuestas se encuentran después de un tiempo determinado.

d₁ = v₁ * t
d₂ = v₂ * t
d₁ + d₂ = distancia total
