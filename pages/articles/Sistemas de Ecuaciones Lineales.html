1. ¿Qué es un Sistema de Ecuaciones Lineales?

Definición

Conjunto de ecuaciones lineales

Un sistema de ecuaciones lineales es un conjunto de dos o más ecuaciones con las mismas variables que deben satisfacerse simultáneamente.

2x + 3y = 8
1x - 2y = -1

Solución: Los valores de las variables que hacen verdaderas todas las ecuaciones simultáneamente.

2. Clasificación de Sistemas

Compatible

Sistema Compatible Determinado

Tiene una única solución. Las ecuaciones representan rectas/planos que se intersecan en un solo punto.

Solución única: (1, 2)
Compatible

Sistema Compatible Indeterminado

Tiene infinitas soluciones. Las ecuaciones representan la misma recta/plano (son linealmente dependientes).

Solución: Familia de soluciones
Incompatible

Sistema Incompatible

No tiene solución. Las ecuaciones representan rectas/planos paralelos que nunca se intersecan.

No existe solución

Método de Sustitución

1. Despeja una variable en una ecuación.

2. Sustituye esa expresión en la otra ecuación.

3. Resuelve la ecuación resultante para la otra variable.

4. Sustituye el valor encontrado para hallar la primera variable.

Método de Eliminación (Reducción)

1. Multiplica las ecuaciones por constantes para igualar coeficientes.

2. Suma o resta las ecuaciones para eliminar una variable.

3. Resuelve la ecuación resultante.

4. Sustituye el valor encontrado en una ecuación original.

5. Ejemplo Completo: Problema de Mezclas

🧪 Problema: Mezclas en una fábrica de productos químicos

Enunciado: Una fábrica produce dos tipos de solución química. La solución A contiene 40% de ácido nítrico y 60% de agua. La solución B contiene 20% de ácido nítrico y 80% de agua. Se necesitan 100 litros de una solución que contenga exactamente 30% de ácido nítrico. ¿Cuántos litros de cada solución se deben mezclar?

Paso 1: Definir variables

Sea x = litros de solución A

Sea y = litros de solución B

Paso 2: Plantear ecuaciones

Ecuación 1 (volumen total): x + y = 100

Ecuación 2 (ácido nítrico): 0.40x + 0.20y = 0.30 × 100 = 30

Paso 3: Resolver el sistema

De la primera ecuación: y = 100 - x

Sustituyendo en la segunda: 0.40x + 0.20(100 - x) = 30

0.40x + 20 - 0.20x = 30

0.20x = 10

x = 50

Entonces: y = 100 - 50 = 50

Paso 4: Verificar solución

Volumen total: 50 + 50 = 100 ✓

Ácido nítrico: 0.40×50 + 0.20×50 = 20 + 10 = 30 ✓

✅ Solución:

Se deben mezclar 50 litros de solución A y 50 litros de solución B.

📚 Resumen Pedagógico

📐

Interpretación Geométrica

En 2D: Cada ecuación representa una recta. La solución es el punto de intersección.

🔍

Determinante Clave

Para sistema 2×2: Si det(A) ≠ 0 → Solución única. Si det(A) = 0 → Infinitas o ninguna.

⚡

Método Recomendado

Para sistemas pequeños: Sustitución o eliminación. Para sistemas grandes: Métodos matriciales.
